~CSS Attributes Reference~

Attribute Name Description Example Value
color Color Defines what foreground color an HTML element should have. blue
font-size Font Size Defines what font size an HTML text element should render in. Common unit is pixels (px). 12px
width Width Defines what width an HTML element should render in. Common unit is pixels (px). 50px
height Height Defines what height an HTML element should render in. Common unit is pixels (px). 70px
font-family Font Family Defines what text font an HTML text element should render in. Courier
font-weight Font Weight Defines what font weight an HTML text element should render in. Common unit is regular numbers. 400
